About MemGUI

MemGUI is the best Memcached GUI on the market, built for advanced management with secure connections, dump import/export, encrypted local storage, and built-in key indexing. It is the only Memcached GUI that brings these capabilities together in one desktop app, giving you visibility, control, and security without sacrificing Memcached performance. Manage connections, switch between servers, and keep every key, stat, and connection in view.

Version 1.2.0 (stable) adds SSH connections, storage encryption, dump import/export improvements, and faster key indexing with stale key pruning.

What is new in 1.2.0

Released 2026-01-16. Stable release highlights.

Added

  • SSH connections with an improved connection flow.
  • Dump import/export and data import improvements.
  • Storage encryption for local data (required for SSH).
  • Lightweight index update listener for new keys.

Changed

  • Key index architecture sharded with centralized reserved keys.
  • Index reuse and stale key pruning for faster key lookups.
  • Cachedump processing reuses results and reads full slabs.
  • Search limits and data size display refined in the UI.
  • UI polish across menus, confirmation, and dump modals.

Fixed

  • Infinite loading when selecting the active connection.
  • Key count no longer includes reserved keys.

Security

  • Security limitations and optimizations applied across the app.

Advanced Memcached GUI Features

Advanced Management

Run Memcached with full visibility across keys, stats, and connections.

SSH Connections

Securely connect to remote Memcached servers through SSH with host key verification.

Encrypted Storage

Protect local data and SSH secrets with storage encryption.

Dump Import/Export

Export dumps for backup and import data back into a server.

Built-in Key Indexing

Built-in indexing and server-side search with stale key pruning for fast lookups.

SASL Authentication

Secure Memcached connections with username and password when required.

Bulk Key Cleanup

Remove all keys on the current server in one action, with clear confirmations.

Server Insights

Track key counts, uptime, memory usage, cache metrics, and performance trends.

Multi-language UI

Use Portuguese (Brazil), English, and other locales with a selector.
