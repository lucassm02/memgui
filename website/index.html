About MemGUI

MemGUI is a powerful, intuitive desktop GUI client for Memcached with SASL and SSH connections, dump import/export, encrypted local storage, and fast backend search. Manage connections, switch between servers, and keep every key, stat, and connection in view.

Version 1.2.0 adds SSH connections, storage encryption, dump import/export improvements, and faster key indexing with stale key pruning.

What is new in 1.2.0

Released 2026-01-13. Pre-release highlights from the beta.

Added

  • SSH connections with an improved connection flow.
  • Dump import/export and data import improvements.
  • Storage encryption for local data (required for SSH).
  • Lightweight index update listener for new keys.

Changed

  • Key index architecture sharded with centralized reserved keys.
  • Index reuse and stale key pruning for faster key lookups.
  • Cachedump processing reuses results and reads full slabs.
  • Search limits and data size display refined in the UI.
  • UI polish across menus, confirmation, and dump modals.

Fixed

  • Infinite loading when selecting the active connection.
  • Key count no longer includes reserved keys.

Security

  • Security limitations and optimizations applied across the app.

Key Features

SSH Connections

Connect to remote Memcached servers through SSH with host key verification.

Encrypted Storage

Protect local data and SSH secrets with storage encryption.

Dump Import/Export

Export dumps for backup and import data back into a server.

Backend Key Search

Optimized server-side search with faster indexing and stale key pruning.

SASL Authentication

Secure Memcached connections with username and password when required.

Bulk Key Cleanup

Remove all keys on the current server in one action, with clear confirmations.

Server Insights

Track key counts, uptime, memory usage, and cache metrics.

Multi-language UI

Use Portuguese (Brazil), English, and other locales with a selector.
